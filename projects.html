Projects

Everything listed below is a project that was for a class, work, or done on personal time. All are in varying states of completion. Not everything works perfectly.

Formula Fields

A domain specific programming language designed for integration into iSENSE. It's mostly done, and only needs a couple more modules added to the standard library and some tests to be considered complete. Formula Fields was written entirely in Ruby, and makes extensive use of RLTK (Ruby Language Toolkit).

Movie Recommendations

An application for recommending movies to users. It was written for 91.422 (Machine Learning), and was done in conjunction with Misael Moscat and Michael Stowell. My contribution to this project was Python code for extracting, preprocessing and matching movie titles, ratings and tags across several different data sources. Collaborative filtering was then run on this data to produce recommendations.

Better Colorpicker

A color picker UI widget designed for integration into iSENSE. It is available as a Ruby gem. It is written in Coffeescript and CSS.

Better Date/Time Picker

A color picker UI widget designed for integration into iSENSE. It is available as a Ruby gem. It is written in Coffeescript and CSS.

WikiPath

An application written in Racket for 91.301 (Organization of Programming Languages). It finds several semi-optimal paths between any two arbitrary Wikipedia articles. Due to a breaking change in a recent version of Racket, the application is being rewritten in Rust.
